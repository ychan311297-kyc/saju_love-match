# LOVE MATCH 점수 산정 방식

## 최종 점수

각 항목을 0~100점으로 계산한 뒤 아래 비중으로 합산합니다.

```text
최종 점수
= 연애 십신 × 0.35
+ 일지 궁합 × 0.20
+ 천간합 × 0.15
+ 오행 밸런스 × 0.15
+ 신강·신약 보완 × 0.10
+ 시주 궁합 × 0.05
```

## 1. 연애 십신 — 35%

| 십신 | 점수 |
|---|---:|
| 정재 | 100 |
| 정관 | 98 |
| 편재 | 94 |
| 편관 | 90 |
| 정인 | 84 |
| 식신 | 82 |
| 비견 | 72 |
| 편인 | 63 |
| 상관 | 60 |
| 겁재 | 48 |

## 2. 일지 궁합 — 20%

| 관계 | 점수 |
|---|---:|
| 육합 | 100 |
| 삼합 | 91 |
| 방합 | 79 |
| 비화 | 74 |
| 평 | 60 |
| 해 | 37 |
| 충 | 20 |

충은 장기 안정에는 불리하지만 강한 긴장감이나 끌림이 생길 수 있어
별도의 설렘 점수에서는 추가 보정을 받습니다.

## 3. 천간합 — 15%

- 일간 천간합 성립: 100
- 천간합 없음: 55

천간합 조합:

- 甲–己
- 乙–庚
- 丙–辛
- 丁–壬
- 戊–癸

## 4. 오행 밸런스 — 15%

사용자 사주의 부족한 오행을 상대의 년·월·일주가 보완하면 점수가 올라갑니다.
이미 과한 오행을 상대가 더 강화하면 점수가 내려갑니다.

상대의 출생 시각은 공개되지 않은 경우가 많아 사용하지 않습니다.

## 6. 시주 궁합 — 5%

사용자의 시지와 상대의 일지를 비교합니다.

## 설렘 점수

```text
설렘
= 연애 십신 × 0.40
+ 천간합 × 0.30
+ 일지 궁합 × 0.20
+ 시주 궁합 × 0.10
```

편재·편관·상관 관계는 설렘에 추가 가점을 받고,
일지 충 관계도 긴장감 보정이 추가됩니다.

## 안정 점수

```text
안정
= 일지 궁합 × 0.35
+ 연애 십신 × 0.25
+ 오행 밸런스 × 0.20
+ 신강·신약 보완 × 0.15
+ 시주 궁합 × 0.05
```

## 해석 범위

이 점수표는 정통 명리학의 표준 궁합 공식을 그대로 구현한 것이 아닙니다.
명리 요소를 유사연애 덕질이라는 목적에 맞게 재구성한 오락용 모델입니다.
