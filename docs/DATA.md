# 데이터 추가 가이드

데이터 원본은 프로젝트 루트의 `idols.json`입니다.

## 필드

| 필드 | 설명 | 예시 |
|---|---|---|
| `name` | 화면에 표시할 이름 | `강유찬` |
| `group` | 그룹명 또는 활동 구분 | `A.C.E` |
| `agency` | 소속사 또는 분류 | `기타` |
| `gender` | 남성 `M`, 여성 `F` | `M` |
| `dob` | 양력 생년월일 | `1997-12-31` |
| `cat` | 카테고리 | `K-idol` |

## 예시

```json
[
  {
    "name": "강유찬",
    "group": "A.C.E",
    "agency": "기타",
    "gender": "M",
    "dob": "1997-12-31",
    "cat": "K-idol"
  },
  {
    "name": "스즈키 노부유키",
    "group": "스즈키 노부유키",
    "agency": "기타",
    "gender": "M",
    "dob": "1992-10-14",
    "cat": "J-idol"
  }
]
```

## JSON 문법 주의

- 전체는 `[`와 `]`로 감쌉니다.
- 각 인물은 `{`와 `}`로 감쌉니다.
- 마지막 인물을 제외하고 각 객체 뒤에는 쉼표가 필요합니다.
- 키와 문자열은 큰따옴표를 사용합니다.
- 같은 `name + group` 조합은 중복으로 간주합니다.

## 데이터 검사

```bash
node tools/validate.js
```

검사 항목:

- JSON 문법
- 필수 필드 존재
- 이름과 그룹이 비어 있지 않은지
- 성별이 `M` 또는 `F`인지
- 날짜가 실제 존재하는 날짜인지
- 중복된 `name + group`
